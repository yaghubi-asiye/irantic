# irantic



## Getting started

For run locally please flow these steps

Step 1:
```
git clone https://github.com/yaghubi-asiye/irantic.git
```
Step 2:
```
cd irantic
composer update
```

Step 3:
```
php artisan migrate --seed

```

Step 4:
```
php artisan jwt:secret

php -S localhost:8000 -t public
```
